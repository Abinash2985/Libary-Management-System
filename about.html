About Library Pro

Library Pro is a student-ready library management system built with HTML, CSS, and JavaScript.

Client-side persistence

All records are stored locally using browser LocalStorage so the app works without a backend.

Feature-packed

Includes book and student CRUD, issue & return workflows, overdue fines, search, filters and reports.

Responsive & modern

A polished layout with cards, tables, modals, transitions and responsive navigation.

Build your project
